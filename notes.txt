Apache Notes (Ubuntu 18):
   Files Stored at: /var/www/html/

Set Permissions:
   1. Find out name of apache user: ps aux | egrep '(apache|httpd)'
   2. Set permissions for data folder, and db file
	- chown -R <user>:<group> <db file>
	- chmod -R u+w <db file>

Docker commands to know:
   Build:
     - create an image from a Dockerfile: docker build <dockerfile_path>
	- options for build: -t <tag>
   Run:
     - run image with bash: docker run -it <image> bash
     - run image with ports: docker run -p <port> <image>
     - terminal for container: docker exec -it <container> bash
   Images:
     - list all images: docker images -a
     - remove an image: docker rmi <image>
   Containers
     - list all containers: docker container ls -a
     - stop a container: docker stop <container>
     - remove a container: docker rm <container>

How to install scp:
   - Update apt-get: apt-get update -y
   - Install: apt-get install openssh-client
